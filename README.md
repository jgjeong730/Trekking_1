# My Trekking Planner - 지리산 화대종주 트레킹 마스터

이 앱은 트레킹 일정 관리, 실시간 체크, 장비 준비, 비용 정산을 한 앱에서 관리할 수 있는 PWA(Progressive Web App)입니다.

## 주요 기능
- **코스 개요**: 화대종주 SVG 루트 맵 및 전체 일정/비용 요약
- **상세 일정**: 일자별/구간별 상세 타임라인 (교통/식사/등반/숙박 색상 구분)
- **실시간 체크**: 현재 진행률, 누적 이동 거리 및 남은 거리 자동 계산
- **장비 체크리스트**: 카테고리별 장비 관리 및 필수도별 배지 표시
- **비용 정산**: 일자별 항목별 지출 합계 관리
- **오프라인 모드**: PWA 적용으로 산속에서도 안정적으로 작동 (LocalStorage 저장)

## 기술 스택
- React
- Tailwind CSS
- Lucide React (Icons)
- Framer Motion (Animations)
- Vite (Build Tool)

## 시작하기
```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 🏔️ 상세 코스 데이터 (NotebookLM 참고용)

이 섹션은 NotebookLM 등 AI 모델이 코스 데이터를 학습하기 위한 상세 텍스트입니다.

### [코스 1] 지리산 화대종주
- **전체 요약**: 화엄사에서 대원사까지 지리산 주능선을 종주하는 코스 (54.6km)
- **총 예상 비용**: 200,300원 (1인 기준)

#### 1일차: 화엄사 → 노고단 (누적 8km)
- **교통**: 서울남부(09:00) → 구례(12:10) [버스, 29,000원]
- **식사**: 구례 점심 (10,000원)
- **이동**: 구례 → 화엄사 일주문 [택시, 20,000원]
- **등반**: 화엄사(13:20) → 노고단 대피소(17:20) [7km]
- **관람**: 노고단 고개 일몰 (18:29)
- **숙박**: 노고단 대피소 (22,000원 - 식사 포함)

#### 2일차: 노고단 → 연하천 (누적 10.2km+)
- **일출**: 노고단 정상 (06:54)
- **경유지**: 돼지령, 피아골삼거리, 임걸령, 노루목, 반야봉(1,732m), 삼도봉, 화개재, 토끼봉
- **숙박**: 연하천 대피소 (22,000원 - 식사 포함)

#### 3일차: 연하천 → 장터목 (누적 11.4km+)
- **경유지**: 벽소령, 세석평전, 촛대봉, 연하봉, 제석봉
- **메인**: 천왕봉(1,915m) 인증 (16:40)
- **숙박**: 장터목 대피소 (23,000원 - 식사 포함)

#### 4일차: 장터목 → 대원사 → 서울 (누적 13.4km)
- **일출**: 천왕봉 일출 (06:51)
- **경유지**: 중봉, 써리봉, 치밭목 대피소, 무제치기폭포, 유평, 대원사
- **귀가**: 대원사 → 산청터미널 → 서울남부(19:35) [약 45,000원 지출]
